# B-Safe - Safety Management Platform

A comprehensive web-based safety management platform designed to help businesses and individuals maintain the highest safety standards.

## Project Structure

```
B-Safe/
├── index.php           # Main website page
├── css/
│   └── style.css      # Main stylesheet
├── js/
│   └── script.js      # JavaScript functionality
└── README.md          # Project documentation
```

## Features

- ✅ Advanced Security - Enterprise-grade security measures
- ✅ Real-time Monitoring - Monitor safety metrics in real-time
- ✅ Instant Alerts - Get notifications for critical events
- ✅ Mobile Friendly - Responsive design for all devices
- ✅ Team Collaboration - Seamless team collaboration features
- ✅ Easy Integration - Simple integration with existing systems

## Getting Started

### Installation

1. Place the B-Safe folder in your XAMPP htdocs directory
2. Start Apache and MySQL from XAMPP Control Panel
3. Open your browser and navigate to: `http://localhost/B-Safe/`

### Requirements

- Apache Web Server (via XAMPP)
- PHP 7.4 or higher
- Modern web browser (Chrome, Firefox, Safari, Edge)

## File Descriptions

### index.php
The main landing page containing:
- Navigation bar with smooth scrolling
- Hero section with call-to-action
- Features showcase grid
- About section
- Contact form
- Footer

### css/style.css
Contains all styling for:
- Responsive layout (mobile-first design)
- Modern gradient themes
- Smooth animations and transitions
- Typography and spacing
- Grid layouts for features

### js/script.js
Provides functionality for:
- Smooth scroll navigation
- Contact form handling
- Intersection Observer for animations
- Event listeners for user interaction
- Console logging for debugging

## Customization

### Colors
Edit the gradient colors in `css/style.css`:
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### Content
Modify the text and icons in `index.php` to match your specific safety platform needs.

### Features
Add or remove feature cards by editing the HTML grid in the Features section.
